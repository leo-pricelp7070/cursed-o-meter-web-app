## What Cursed-O-Meter Is

Cursed-O-Meter is a browser-friendly HTML project that presents random facts and couples each one with a harmlessness rating. It is meant to feel like a straightforward interactive webpage instead of a heavy application, so it is easy to open, browse, and show without any extra setup.

This makes it a handy option for anyone looking for a compact fact viewer with a playful scoring twist. The project also has a distinct look thanks to its custom visual assets, which were created in Affinity Designer and help define the page's style.

## How to Use It

Load the webpage, then refresh or interact with the interface to surface a new random fact and its harmlessness rating. The experience is intentionally simple, so the usual flow is just open the page, inspect the fact, and repeat.

If you are hosting it locally, browse to the running page and keep the files in the same folder layout so the icons and images can load as expected.

## Configuration

Most of the behavior lives in the HTML and nearby front-end files. To change how it feels, edit the page copy, fact list, rating logic, or image paths in the source.

A simple setup often looks like this:

{
  "facts_source": "embedded content or linked data",
  "rating_display": "harmlessness score shown in the UI",
  "assets": "custom icons and images"
}
